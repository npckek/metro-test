# API Станций Метро

## Описание
RESTful API для обработки и предоставления точных геокоординат и статистики станций метро.

## Создание секретов
Создайте файл .env и задайте необходимые переменные:
```bash
cp .env.example .env
```

## Запуск
Требуется Docker и Docker Compose.
```bash
docker compose up --build
